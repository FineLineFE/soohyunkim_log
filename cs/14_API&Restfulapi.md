## API
``` 
통신에 사용되는 메시지 형식. application programming interface.
```
라이브러리가 제공하는 여러 함수를 이용하여 프로그래밍할 때 api에 단순히 입력값을 주고 결과값을 사용할 수 있게 한다.

주방 = 라이브러리
메뉴 = api
직원 = 연결

## Restful API
```
쉽게 말해 엔드포인트의 주소를 결정하는 일종의 규약된 패턴. http 메소드(CRUD)/명사 구조.
```

일관성을 유지하기 위해 사용된다.

- Post/user
- Get/user
- Delete/user
